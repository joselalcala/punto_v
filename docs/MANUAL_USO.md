# Manual de Uso Rápido

## 1. Ingreso

- Abrir la aplicación
- Iniciar sesión con un usuario activo

## 2. Configuración inicial

- Ir a `Empresa`
- Definir moneda, impuesto, stock mínimo y métodos de pago habilitados
- Verificar comprobantes disponibles

## 3. Apertura de caja

- Entrar a `Cajas`
- Crear una nueva caja
- Elegir ubicación si aplica
- Registrar saldo inicial

## 4. Compra

- Ir a `Compras`
- Seleccionar proveedor, comprobante y método de pago
- Agregar productos al detalle
- Confirmar compra

## 5. Venta

- Ir a `Ventas`
- Seleccionar cliente, comprobante y método de pago
- Agregar productos
- Confirmar el cobro

## 6. Caja y movimientos

- Entrar a movimientos de la caja abierta
- Revisar ventas, retiros y balance neto
- Registrar retiros cuando sea necesario
- Cerrar caja al finalizar

## 7. Reportes operativos

- Consultar el panel principal para métricas rápidas
- Filtrar ventas y compras por fecha y método de pago
- Exportar ventas a Excel
- Abrir comprobantes PDF
